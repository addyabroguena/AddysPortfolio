Inventory Management System

A simple ATM-style inventory management system built using Java. This project was developed as part of an exam to demonstrate essential programming logic and user interaction in a console-based environment.

The system allows users to store data in a dynamic structure, add new inventory items with their name, price, and quantity, update item quantities, and display all inventory records in a readable format. By simulating basic inventory operations, the project highlights fundamental concepts in object-oriented programming, such as classes, methods, and data encapsulation.

Through this exercise, I gained hands-on experience with control structures, user input handling, and real-time data manipulation using Java collections.

Technologies: JavaScript

Contact Book

A feature-rich contact book application designed to manage and organize contacts into user-defined groups. This project allows users to add or remove contacts from groups, view contacts by category, and perform advanced searches using criteria like partial names, phone numbers, or group filters.

The system supports importing and exporting contact data in CSV format, making it easy to back up or transfer information. Nested data structures were used to represent the relationships between groups and contacts, enabling efficient data management. Robust file handling and error checking ensure reliability when reading from or writing to external files.

The result is a practical, console-based contact management system that demonstrates strong problem-solving, file handling, and data structuring capabilities.

Technologies: JavaScript
